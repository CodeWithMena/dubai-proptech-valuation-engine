# Dubai Real Estate Investment Analysis



## Project Overview



This project analyzes Dubai’s off-plan property market to help investors identify which neighborhoods offer the best rental returns. Using 12,000 property transaction records, I cleaned the data, removed outlier valuations, and calculated average property prices and gross annual rental yields across five major development areas.



## Tools Used



- Python (Pandas, NumPy) — Google Colab

- Dataset: Dubai off-plan property transactions



## Business Question



Which Dubai neighborhoods offer the highest rental yield relative to property price?



## Data Cleaning



- Removed placeholder entries below AED 10,000

- Standardized property area units

- Filtered and mapped five highest-volume development corridors

## Business Recommendation



Investors prioritizing rental income over capital appreciation should consider Marina Vista and Seaside — both deliver above 7.8% gross annual returns based on current transaction data.



Grande commands the highest property prices but delivers moderate yield, suggesting it attracts capital appreciation investors rather than rental income seekers.
